Introduction

{{.PageTitle}} dans ce tutoriel sur le nouveau langage de programmation basé sur un moteur MVC.

Dans ce langage, les routes sont chargées depuis un fichier spécifique appelé routes.bird.

Chargement des routes HTML

Dans le fichier routes.bird, vous pouvez définir les différentes routes de votre application.

Voici un exemple de syntaxe pour définir des routes :

birdfiles/routes.bird:

route("/")
    return "index.html"

route("/contact")
    return "contact.html"

route("/faq")
    return "faq.html"

Dans cet exemple, chaque route est associée à une URL et retourne le nom du fichier HTML correspondant.

Exemples de fichiers HTML

Voici quelques exemples de fichiers HTML correspondant aux routes définies dans routes.bird:

  • index.html - Page d'accueil de l'application.
  • contact.html - Page de contact.
  • faq.html - Page des questions fréquemment posées.
